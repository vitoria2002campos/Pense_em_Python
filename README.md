O que é um programa?
É uma sequencia de instruções que chegam em um resultado especifico. 
Na programação temos algumas intruções básicas que aparecem em quase todas as linguagens:
Entrada: Receber dados do teclado, de um arquivo, da rede ou de algum outro dispositivo
Saida: saida dos resultados do arquivo
